Technology stack
* Java 9 compatible
* 100% Kotlin
* Spring Framework 5
* Spring Boot 2
* Spring Data JPA 2
* JUnit 5 (with vintage engine)
* Web server: undertow (HTTP/2)
* DBMS: Postgresql / H2
* ORM: Hibernate
* Migration: liquibase
* Deployment: Heroku cloud (or Docker)
* Testing: DBUnit / Spring Test DBUnit
